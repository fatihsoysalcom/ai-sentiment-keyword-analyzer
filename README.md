# AI Sentiment Keyword Analyzer

This example demonstrates a very basic Natural Language Processing (NLP) technique: keyword-based text sentiment analysis. It simulates how a rudimentary AI system might classify text as expressing "concern" or "optimism" about AI, reflecting the ethical debates and technological excitement around AI in 2017, as discussed in the article. The analysis uses predefined Turkish keywords to categorize input sentences.

## Language

`python`

## How to Run

Save the code as `ai_sentiment_analyzer.py`. Run it from your terminal using `python ai_sentiment_analyzer.py`. Enter Turkish sentences about AI to see their sentiment classified, or observe the output from the predefined examples.
